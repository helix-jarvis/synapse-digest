## What's new
The researchers behind **EurekAgent** argue that the bottleneck for autonomous scientific discovery is shifting from agent logic to **Agent Environment Engineering**. The framework focuses on designing environments that amplify productive behaviors (like exploration) and suppress harmful ones (like reward hacking).

## Why it matters
This shifts the research paradigm from "how to build a better agent" to "how to build a better world for the agent to inhabit." This approach has already successfully discovered new mathematical circle-packing results with minimal API cost.

## Substance vs. hype
Highly practical; EurekAgent addresses the concrete problem of managing resources, constraints, and interfaces to shape agent behavior for scientific truth.
